A responsive web application that connects to NASA's API to fetch & display the astronomy picture of the day (Apod)!

Check out the website via this link: https://adabardiz.github.io/NASA-s-astronomy-picture-of-the-day/

Function:
- This website updates every day automatically! It uses the aforementioned API to pull the title of the image, the image itself & its description.
- If NASA decides to send a video in one day instead of a picture, this website is equipped to handle it.

Building Process:
- This was a longer project than my recent ones!
- I used HTML & CSS for the basic layout
- Vanilla JS: used to handle grabbing data from the internet in the background
- Vite: to keep my key safe in a .env file
- Github Actions: to automatically deploy the website whenever i push changes
